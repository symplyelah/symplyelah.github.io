# Data Talks

This [site](https://ifydatatalks.netlify.app/) is a hub for my data-focused talks, workshops, and presentations. Whether you’re a student, professional, or simply curious about data, you’ll find practical insights, downloadable slides, and real-world examples to support your journey.

I believe that data should be clear, accessible, and meaningful — and that anyone can learn to work with it.

Take a look around, explore my latest sessions, and feel free to reach out if you’d like to connect or collaborate!
